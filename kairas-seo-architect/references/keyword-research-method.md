# Método de Keyword Research para KAIRAS

## Proceso completo de investigación de keywords

### Paso 1 — Definir el tema y el objetivo

Antes de buscar keywords, define:
- ¿Para qué tipo de página es? (servicio, blog, landing local, categoría)
- ¿Qué servicio de KAIRAS se quiere posicionar?
- ¿A qué sector va dirigido? (clínicas, academias, pymes genéricas, etc.)
- ¿Tiene intención geográfica? (Galicia, Pontevedra, Vigo, etc.)
- ¿Qué etapa del embudo cubre? (descubrimiento, evaluación, decisión)

### Paso 2 — Generar la semilla de keywords

Parte de una lista inicial de keywords semilla relacionadas con el tema:

**Para KAIRAS, los grandes clusters semilla son:**

```
Cluster A — Automatización + zona
  automatización empresas Galicia
  automatizar procesos Pontevedra
  automatización IA Vigo
  agencia automatización Galicia

Cluster B — IA para empresas / pymes
  IA para empresas Galicia
  inteligencia artificial pymes
  IA para pymes de servicios
  IA aplicada negocios

Cluster C — Automatización por canal
  automatización WhatsApp empresas
  bot WhatsApp para clínicas
  respuesta automática WhatsApp negocio
  automatización email seguimiento

Cluster D — CRM y seguimiento
  automatización CRM pymes
  seguimiento automático clientes
  CRM para clínicas
  recordatorios automáticos clientes

Cluster E — Por sector
  automatización para clínicas
  IA para academias
  automatización inmobiliaria
  IA para clínicas estéticas
  
Cluster F — Optimización operativa
  reducir tareas manuales empresa
  ordenar procesos empresa
  optimización operativa pymes
  integrar herramientas empresa
```

### Paso 3 — Expandir con variaciones

Para cada keyword semilla, generar variaciones:

**Variaciones por formato de búsqueda:**
- ¿Cómo + [hacer algo]? → "cómo automatizar los recordatorios de citas"
- ¿Cuánto cuesta + [servicio]? → "cuánto cuesta automatizar una clínica"
- ¿Qué es + [concepto]? → "qué es la automatización de procesos"
- ¿Cuál es el mejor + [herramienta/método]? → "cuál es el mejor CRM para clínicas"
- [Concepto] + para + [tipo de empresa] → "automatización para academias"
- [Concepto] + en + [zona] → "automatización de procesos en Pontevedra"

**Variaciones por sinónimo o término relacionado:**
- automatización = automatizar, automatizado, sistema automático
- IA = inteligencia artificial, AI, IA aplicada
- pymes = pequeñas empresas, autónomos, negocios
- Galicia = gallego, gallega, noroeste de España

**Variaciones long-tail:**
- "cómo automatizar el seguimiento de clientes en una clínica estética"
- "qué herramientas usar para automatizar WhatsApp en una pyme"
- "cómo ordenar los procesos de una academia sin contratar más personal"

### Paso 4 — Clasificar por intención de búsqueda

| Intención | Señales en la query | Acción esperada del usuario | Tipo de contenido |
|-----------|---------------------|----------------------------|-------------------|
| Informacional | qué es, cómo, para qué, cuándo, guía, tipos de | Aprender | Blog post, guía |
| Comercial | mejor, comparar, opciones, herramientas para, agencia de | Evaluar | Comparativa, servicio |
| Transaccional | contratar, precio, presupuesto, solicitar | Contactar/comprar | Landing, contacto |
| Local | en Galicia, en Vigo, cerca, en Pontevedra | Encontrar proveedor local | Landing local, servicio+zona |
| Comparativa | vs, diferencia, cuál es mejor | Decidir entre opciones | Post comparativo |

**Regla:** Una página = una intención. No mezclar informacional y transaccional en el mismo contenido de forma forzada.

### Paso 5 — Priorizar keywords

**Criterios de priorización (sin herramientas):**

| Criterio | Peso | Razonamiento |
|----------|------|-------------|
| Valor comercial para KAIRAS | Alto | ¿Trae clientes potenciales reales? |
| Dificultad estimada (baja = mejor) | Alto | ¿La competencia en SERP es débil o fuerte? |
| Intención cualificada | Alto | ¿El buscador tiene perfil de cliente? |
| Volumen estimado | Medio | Más búsquedas = más potencial, pero no es lo único |
| Sintonía con la marca | Medio | ¿Es un tema que KAIRAS puede cubrir con autoridad? |
| Urgencia estratégica | Variable | ¿Necesita KAIRAS visibilidad urgente en este tema? |

**Con herramientas, añadir:**
- Volumen mensual real (Search Console, Keyword Planner, Semrush, Ahrefs)
- Dificultad de keyword (KD en Ahrefs/Semrush)
- CPC como indicador de valor comercial (más CPC = más intención de compra)
- SERP features disponibles (featured snippet, PAA, mapas locales)
- Tendencia (Google Trends): ¿creciente, estable o bajando?

**Escala de prioridad sugerida:**

| Prioridad | Criterio |
|-----------|---------|
| Alta | Valor comercial alto + dificultad potencialmente asumible + intención cualificada |
| Media | Valor comercial medio O dificultad alta pero keyword importante para autoridad temática |
| Baja | Informacional puro sin conexión comercial, volumen muy bajo o intención no cualificada |
| Descartar | Tráfico basura claro: lectores sin perfil de cliente, keywords demasiado genéricas |

### Paso 6 — Detectar canibalización

La canibalización ocurre cuando dos o más páginas del sitio compiten por la misma keyword, confundiendo a Google sobre cuál priorizar.

**Cómo detectarla:**
- Revisar si ya existe una página en kairas.es que cubre ese tema/keyword
- Buscar en Google: `site:kairas.es [keyword]` — si aparecen 2+ páginas, hay riesgo
- Revisar el mapa de keywords existente para detectar solapamientos

**Soluciones:**
- Si dos páginas son similares: fusionarlas en una con redirect 301
- Si cubren ángulos distintos: diferenciar claramente la intención de cada una y los títulos
- Si una es mucho más fuerte: apuntar los enlaces internos solo a la que debe ganar
- Si son del mismo silo: asegurarse de que una es la pilar y la otra es satélite con diferente ángulo

### Paso 7 — Alertas de tráfico basura

Keywords que parecen atractivas para KAIRAS pero pueden atraer usuarios sin perfil:

| Keyword | Por qué puede ser basura | Alternativa mejor |
|---------|--------------------------|-------------------|
| "qué es la IA" | Curiosidad genérica, no busca contratar | "IA para pymes" o "IA aplicada a negocios" |
| "herramientas de automatización gratis" | Busca soluciones DIY, no agencia | "herramientas de automatización para empresas" |
| "chatGPT para empresas" | Busca usar ChatGPT él mismo | "IA aplicada para pymes de servicios" |
| "automatización de marketing" | Perfil más de marketero que de empresario pyme | "automatización para clínicas" o "para academias" |
| "n8n tutorial" | Desarrolladores o curiosos técnicos | "automatización sin programar para pymes" |
| "zapier gratis" | Busca herramienta DIY | Sin alternativa directa, ignorar |
